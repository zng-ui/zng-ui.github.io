where T: VarValue,

New with a single value thumb of type T.

The value must convert to a normalized factor [0.fct()..=1.fct()] where 0.fct() is the minimum possible value and 1.fct() is the maximum possible value. If a value outside of this range is returned it is clamped to the range and the selection variable is updated back.
