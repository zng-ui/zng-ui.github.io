Struct ContextInitHandle

pub struct ContextInitHandle(/* private fields */);
Expand description

Identifies the unique context a ContextualizedVar is in.

Each node that sets context-vars have an unique ID, it is different after each (re)init. The ContextualizedVar records this ID, and rebuilds when it has changed. The contextualized inner vars are retained when the ID has at least one clone.
