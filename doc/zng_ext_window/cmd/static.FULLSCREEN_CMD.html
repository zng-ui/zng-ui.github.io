pub static FULLSCREEN_CMD: Command

Represents the window toggle fullscreen action.

§Behavior

This command is about the windowed fullscreen state (WindowState::Fullscreen), use the EXCLUSIVE_FULLSCREEN_CMD to toggle exclusive video mode fullscreen.

§Metadata

This command has the following default metadata:

metadatavalue
name "Fullscreen"
info "Toggle fullscreen mode on the window."
shortcut { let a = if cfg!(target_os = "macos") { shortcut!(CTRL|SHIFT+'F') } else { shortcut!(F11) }; [a, shortcut!(ZoomToggle)] }
icon wgt_fn!(|_| ICONS.get("fullscreen"))

Text metadata is localized.
