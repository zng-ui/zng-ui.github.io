Struct AnchorOffset 

Source 
pub struct AnchorOffset {
    pub place: Point,
    pub origin: Point,
}
Expand description

Represents two points that position a layer widget with its anchor widget.

The place point is layout in the anchor widget bounds, the origin point is layout in the layer widget bounds, the layer widget is offset so that the origin point aligns with the place point.
