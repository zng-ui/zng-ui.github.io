P The view generator.

The update widget handler is used to generate the view from the data, it is called on init and every time data or update are new. The view is set by calling DataViewArgs::set_view in the widget function args, note that the data variable is available in DataViewArgs::data, a good view will bind to the variable to support some changes, only replacing the view for major changes.
