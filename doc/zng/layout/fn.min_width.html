Expand description

P Minimum width of the widget.

The widget width can be larger then this but not smaller. Relative values are computed from the constraints maximum bounded width.

This property does not force the minimum constrained width, the min_width is only used if it is greater then the constrained minimum.

This property disables inline layout for the widget.
