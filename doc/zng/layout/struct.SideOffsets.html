Struct SideOffsets

source 
pub struct SideOffsets {
    pub top: Length,
    pub right: Length,
    pub bottom: Length,
    pub left: Length,
}
Expand description

2D size offsets in Length units.

This unit defines spacing around all four sides of a box, a widget margin can be defined using a value of this type.
