Represents the action of scrolling until a child widget is fully visible, the command can also adjust the zoom scale.

ยงMetadata

This command initializes with no extra metadata.

ยงParameter

This command requires a parameter to work, it can be a ScrollToRequest instance, or a ScrollToTarget, or the WidgetId of a descendant of the scroll, or a Rect resolved in the scrollable space.

You can use the scroll_to function to invoke this command in all parent scrolls automatically.
