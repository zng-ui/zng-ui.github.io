Struct Deadline 

Source 
pub struct Deadline(pub DInstant);
Expand description

Represents a timeout instant.

Deadlines and timeouts can be specified as a DInstant in the future or as a Duration from now, both of these types can be converted to this struct.

§Examples

In the example below the timer function accepts Deadline, DInstant and Duration inputs.

fn timer(deadline: impl Into<Deadline>) {
    let deadline = deadline.into();
    // ..
}

timer(5.secs());
