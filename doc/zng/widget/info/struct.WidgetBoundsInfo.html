Struct WidgetBoundsInfo 

Source 
pub struct WidgetBoundsInfo(/* private fields */);
Expand description

Shared reference to layout size, offsets, rendered transforms and bounds of a widget.

Can be retrieved in the WIDGET and WidgetInfo.
