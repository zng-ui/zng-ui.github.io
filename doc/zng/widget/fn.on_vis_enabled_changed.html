Expand description

P Widget changed to enabled or disabled visuals.

Note that this event tracks the visual enabled status of the widget, not the actual status, the widget may still be blocked, see Interactivity for more details.

Note that an event is received when the widget first initializes in the widget info tree, this is because the interactivity changed from None, this initial event can be detected using the is_new method in the args.

See on_interactivity_changed for a more general interactivity event.
