pub static HIT_TEST_MODE_VAR: ContextVar<HitTestMode>

Defines the hit-test mode for a widget and descendants.

This variable can be set using the hit_test_mode property.

Note that hit-test is disabled for the entire sub-tree, even if a child sets to a different mode again, the hit_test_mode property already enforces this, custom nodes should avoid overriding Disabled, as the hit-test will still be disabled, but other custom code that depend on this variable will read an incorrect state.
