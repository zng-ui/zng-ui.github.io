Struct zng_app::widget::base::NonWidgetBase

source · 
pub struct NonWidgetBase { /* private fields */ }
Expand description

Base widget for types that build to a custom type that is not used as a part of the UI tree.

This type can be used as base instead of WidgetBase for types that provide a custom build that outputs an instance that is not used as a widget in the UI tree.
