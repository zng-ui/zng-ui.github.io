pub fn search_index(&self, search: &str) -> Option<usize>

Gets if the setting should be included in the search and how likely it is to be an exact match (0 is exact).

If search starts with @key: matches key case sensitive, otherwise matches name or description in lower case. Note that non-key search is expected to already be lowercase.
