pub fn max_size(child: impl UiNode, max_size: impl IntoVar<Size>) -> impl UiNode
Expand description

P Maximum size of the widget.

The widget size can be smaller then this but not larger. Relative values are computed from the constraints maximum bounded size.

This property does not force the maximum constrained size, the max_size is only used in a dimension if it is less then the constrained maximum, or the maximum was not constrained.

This property disables inline layout for the widget.
