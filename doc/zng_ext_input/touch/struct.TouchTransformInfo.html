Struct TouchTransformInfo

Source 
pub struct TouchTransformInfo {
    pub touches: [Point2D<f32, Px>; 2],
    pub center: Point2D<f32, Px>,
    pub deviation: f32,
    pub deviation_x: f32,
    pub deviation_y: f32,
    pub angle: AngleRadian,
}
Expand description

Info useful for touch gestures computed from two touch points.

Fields§

§touches: [Point2D<f32, Px>; 2]

The two touch contact points.

§center: Point2D<f32, Px>

Middle of the line between the two points.

§deviation: f32

Deviation from the points to the center.

Min 1.0

§deviation_x: f32

Deviation from the points.x to the center.x.

Min 1.0

§deviation_y: f32

Deviation from the points.y to the center.y.

Min 1.0

§angle: AngleRadian

Angle of the line.
