Enum CaptureMode

Source 
pub enum CaptureMode {
    Window,
    Subtree,
    Widget,
}
Expand description

Mouse and touch capture mode.

Variants§

§

Window

Mouse and touch captured by the window only.

Default behavior.

§

Subtree

Mouse and touch events inside the widget sub-tree permitted. Mouse events outside of the widget redirected to the widget.

§

Widget

Mouse and touch events redirected to the widget.
