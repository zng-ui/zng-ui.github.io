Struct RadialGradient

Source 
pub struct RadialGradient { /* private fields */ }
Expand description

Radial gradient.

Can be used as a node that fills the available space with the gradient, or can continue building a radial or tiled radial gradient.

Use gradient or radial_gradient to start building.
