pub static SHORTCUT_SPACING_VAR: ContextVar<Length>
Expand description

Minimum space between a menu item child and the shortcut_txt child.

The spacing is applied only if the shortcut child is set to a non-zero sized widget and there is no other wider menu item.

Is 20 by default.
